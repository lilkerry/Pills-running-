# 💊 Realistic Pills System for FiveM (QBX/QBCore)

A comprehensive and realistic pills/medication system for FiveM servers running QBX or QBCore framework.

## 📋 Features
               KERRYGAMER 
### Core Features
- ✅ **Multiple Pill Types** - Painkillers, Antibiotics, Stimulants, Sedatives, Vitamins, Aspirin
- ✅ **Realistic Effects** - Health restoration, stamina boost, speed enhancement, calming effects
- ✅ **Side Effects System** - Nausea, dizziness, headaches, tremors with configurable chance
- ✅ **Overdose Protection** - Prevents pill abuse with damage and visual effects
- ✅ **Animation System** - Smooth pill-taking animations
- ✅ **Visual Effects** - Screen effects matching pill type
- ✅ **Audio Feedback** - Sound effects for pill consumption
- ✅ **Duration System** - Each pill has configurable effect duration
- ✅ **Addiction System** (Optional) - Can enable addiction mechanics
- ✅ **Server Logging** - Track pill usage with player names and citizen IDs
- ✅ **Export Functions** - Easily integrate with other scripts

### Pill Types

| Pill Type | Duration | Effects | Use |
|-----------|----------|---------|-----|
| Painkillers | 10 min | +25 Health, +15 Stamina | Injury recovery |
| Antibiotics | 20 min | +40 Health, Infection cure | Disease treatment |
| Stimulants | 15 min | +50 Stamina, 20% Speed Boost | Combat/Work boost |
| Sedatives | 5 min | Calm effect, Drowsiness | Sleep/Relaxation |
| Vitamins | 30 min | +15 Health, +25 Stamina, Immunity | Daily health maintenance |
| Aspirin | 8 min | +10 Health, Pain relief | Headaches |
